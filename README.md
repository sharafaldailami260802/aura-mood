# Aura — Mood Analytics

A progressive web app for tracking mood, sleep, and energy. Includes journaling, charts, insights, and offline support.

## Features

- Daily check-ins (mood, sleep, energy)
- Journal with rich text
- Mood trends, sleep analysis, correlations
- Works offline (PWA)
- Install as app on phone or desktop
- Export/backup to JSON

## Live site

After enabling GitHub Pages for this repo, the app will be at:

**https://YOUR_USERNAME.github.io/aura-mood/**

## Run locally

```bash
npx serve .
```

Then open http://localhost:3000

## Tech

Single-page app: HTML, CSS, JavaScript. Data stored in IndexedDB (Dexie). Charts with Chart.js.
